# Knowledge Graph-Based AI Framework for Predicting Nutritional and Health Impacts of Food Ingredients

*November 2025*

---

## 📖 Overview
An AI-powered system that uses **Knowledge Graphs (KG)**, **Graph Neural Networks (GNNs)**, and **Large Language Models (LLMs)** to predict health risks (hypertension, obesity, diabetes), allergic reactions, and nutritional impacts from food ingredients.

---

## 🔑 Key Features
- Real-time knowledge graph construction from USDA FoodData Central, PubMed, FDA recalls  
- GNN-based risk prediction  
- Explainable AI (XAI) with evidence from literature  
- Voice-enabled interface (TTS + speech recognition) for visually impaired users  

---

## ⚙️ Tech Stack
- Python, PyTorch / TensorFlow  
- Neo4j or RDFlib/Owlready2 for Knowledge Graph  
- Graph Neural Networks (PyG, DGL)  
- LLMs (Llama 3, Mistral, etc.) via Ollama/HuggingFace  
- RAG with FAISS/LangChain  
- FastAPI / Streamlit / Gradio frontend  
- Voice: Whisper + TTS
